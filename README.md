# 🎓 Student Portal - React Routing Application

A modern **Student Portal web application** built using **React.js and React Router DOM**.  
This project demonstrates client-side routing, reusable components, and a responsive dashboard-style user interface.

## 🚀 Live Demo

🔗 https://kiruthivasu.github.io/React-Routing/

---

## ✨ Features

- 🏠 Home page with welcome section
- 📖 About page with project information
- 👨‍🎓 Students dashboard to display student details
- 📞 Contact page with contact information
- ❌ Custom 404 Not Found page
- 🔗 Smooth navigation using React Router
- 🧩 Reusable React components
- 🎨 Professional responsive user interface
- ⚡ Fast single-page application experience

## ⚙️ Installation and Setup

### Clone the repository

```bash
git clone https://github.com/Kiruthivasu/React-Routing.git
```

### Navigate to project folder

```bash
cd React-Routing
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

The application will run on:

```
http://localhost:5173
```
